# Backlog Opener

登録済みの Backlog ドメインごとに、課題番号を入力して該当の課題詳細画面を開く Chrome 拡張機能です。

## 使い方

1. Chrome の拡張機能管理画面で「デベロッパーモード」を有効化します。
2. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択します。
3. 拡張機能のオプション画面を開き、Backlog ドメインを 1 行ずつ登録します。
4. 拡張機能アイコンを押すと、登録済みドメインごとの行が表示されます。
5. 開きたい行の課題番号欄に数字を入力し、その行の「開く」ボタンを押します。
6. その行のドメインに対して `https://<domain>/view/<課題番号>` を新規タブで開きます。

## 登録できる値

- `example.backlog.jp`
- `https://example.backlog.com`

スキーム付き・なしのどちらでも入力できます。保存時にはホスト名だけが保持され、重複は自動で除去されます。
